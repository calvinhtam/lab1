Robert Downey Jr. Project : - )


DSC 102 - Data Visualization Project 1

Description:

Here I basically did a fun project about Robert Downey Jr. and his work as an actor.
It pretty much has three main visualizations. The first is a choropleth map of where
his movies were produced. The second is a circle packing graph where the size of the
categories is related to the movies/genres revenue size. The third is a word cloud
of some common words showing up in quick movie descriptions of the movie.
Note that there is some angst in the website because I was quite upset with the
unprofessionalism of the professor and the TAs in how they handled the course.
